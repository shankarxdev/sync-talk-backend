# Spring WebSocket Chat

This is a basic real-time chat application using **Spring Boot** and **WebSocket** (STOMP + SockJS).  
It allows multiple users to join and exchange messages instantly in a simple web interface.

## Features
- Real-time messaging using WebSocket
- Separate chat sessions with unique aliases
- Easy-to-extend for private messaging or chat rooms
- Minimal frontend using HTML, JS, SockJS, and STOMP
- Works with modern browsers

## Tech Stack
- **Backend:** Spring Boot, Spring WebSocket, STOMP
- **Frontend:** HTML, JavaScript, SockJS, STOMP.js
- **Build:** Maven
